### Accesibilidad (WAI/W3C)
- [WAI — Web Accessibility Initiative](https://www.w3.org/WAI/) — Portal oficial de accesibilidad web del W3C
- [WCAG 2.2 — Web Content Accessibility Guidelines](https://www.w3.org/TR/WCAG22/) — Especificación completa de WCAG 2.2
- [WAI-ARIA 1.2](https://www.w3.org/TR/wai-aria-1.2/) — Especificación de Accessible Rich Internet Applications
- [How to Meet WCAG (Quick Reference)](https://www.w3.org/WAI/WCAG22/quickref/) — Referencia rápida de criterios WCAG
- [WAI Tutorials](https://www.w3.org/WAI/tutorials/) — Tutoriales oficiales: imágenes, formularios, tablas, etc.
- [Web Accessibility Evaluation Tools List](https://www.w3.org/WAI/tools/) — Listado oficial de herramientas de evaluación
- [Easy Checks – A First Review](https://www.w3.org/WAI/test-evaluate/preliminary/) — Checklist rápida de accesibilidad
- [WCAG 2.2 en español (traducción autorizada)](https://www.w3.org/Translations/WCAG22-es/) — Traducción al español de las WCAG 2.2
